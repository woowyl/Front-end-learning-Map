# Expression
## Grammer
 - 优先级
 - Member 表达式
    - a.b
    - a[b]
    - foo`string`
    - super.b
    - super['b']
    - new.target
    - new Foo()
 - new 表达式
  - new Foo


- Reference
 - Object key
 - delete assign

- Call 函数调用
    - foo()
    - foo()['a']

- Left handside & Right hand side
有些只能放在等号左边，有些只能放在等号右边
 - a+b = c
 - Right handside
    - a++
    - a--
    - --a
    - a--
- Unay
 -  delete a.b
 - !a
 - void foo()
 - await a
 - + a
