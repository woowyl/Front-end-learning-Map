## vue-cli 
> Vue 提供一个官方命令行工具，可用于快速搭建大型单页应用。该工具为现代化的前端开发工作流提供了开箱即用的构建配置。只需几分钟即可创建并启动一个带热重载、保存时静态检查以及可用于生产环境的构建配置的项目：

``` shell
# 全局安装vue-cli 3新语法
$ npm install -g @vue/cli
# 确认版本是否正确
$ vue --version

```
